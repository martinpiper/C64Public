CharPad 2.7 User Manual - Subchrist Software, 2020.

The Map Editor Form.


The Map Editor form allows you to view and edit the project's map.


Resizing the map...

This form's tool bar contains numeric entry boxes for displaying and changing the map's dimensions, to alter them simply type new values into the boxes and hit ENTER.

nb. The smallest possible map size is 1x1, the largest is 8192 x 8192.


Panning...

With the Pan tool selected, you can navigate around the map by clicking and dragging it with the LEFT mouse button, depending on the map dimensions this is usually quicker and more convenient than using the horizontal and vertical scroll bars.

nb. You can temporarily engage the Pan tool by pressing and holding the SPACE bar.


Selecting a map area...

With the Selection tool enabled, clicking on the map the LEFT mouse button will select a map cell.

By clicking on a second map cell using the LEFT mouse button with the SHIFT key held down, a rectangular area of the map can be selected (it will appear highlighted).

When the Selection tool is active, a number of options become available in the Edit menu such as Select All, Crop, Cut, Copy, Paste, Fill etc.


Painting...

With one of the Brush tools selected, map cells can be painted using the mouse.

The LEFT mouse button paints using the current 'primary' brush and the RIGHT mouse button paints using the current 'secondary' brush.

With the flood-fill tool selected, areas of the map can be flood-filled, clicking the LEFT mouse button while over the drawing canvas will flood-fill from the chosen spot using the current 'primary' brush and the RIGHT mouse button will flood-fill using the current 'secondary' brush.

When using a Brush tool or the Flood-fill tool, the MIDDLE mouse button can be used to pick a new primary brush from the map.


Transformations...

The following transformations are available when this form is active...

• Scroll (up, down, left, right).

• Flip (left-to-right, top-to-bottom).

• Reflect (left-to-right, top-to-bottom).


Notes...

-When the Map form is active, the main form offers the Scroll, Flip and Reflect operations in the Edit menu and also on the tool bar, these can be performed on the whole map, or if the the Selector tool is enabled, these operations can be performed on just a currently selected area.
