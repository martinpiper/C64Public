The Character Image Editor Form.


The Character Image Editor form allows you to edit the currently selected character image.


Painting...

With one of the Pen tools selected, pixels can be drawn using the mouse.

The LEFT mouse button draws using the currently selected colour (on the Project form) and the RIGHT mouse button draws using the background colour.

With the Flood-fill tool selected, areas of the image can be flood-filled from any selected point, clicking the LEFT mouse button while over the canvas will flood-fill using the currently selected colour and the RIGHT mouse button will flood-fill using the background colour.


Transformations...

The following transformations are available when this form is active...

• Scroll (up, down, left, right).

• Flip (left-to-right, top-to-bottom).

• Reflect (left-to-right, top-to-bottom).

• Negative.

• Colour swapping (for multi-colour images).
